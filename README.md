# Klimadaten Analyse Tool

## Beschreibung
Dieses Tool ermöglicht es Benutzern, Klimadaten zu analysieren und zu visualisieren. Es unterstützt den Upload von Datensätzen, bietet verschiedene Analysemöglichkeiten und erstellt interaktive Grafiken zur Darstellung der Ergebnisse.

## Funktionen
- Upload von Klimadaten im CSV-Format
- Verschiedene Analysefunktionen (Statistiken, Trends, etc.)
- Interaktive Grafiken und Diagramme
- Speichern und Exportieren von Analyseergebnissen

## Technologien
- Frontend: React.js
- Backend: Node.js mit Express
- Datenbank: MongoDB
- Visualisierungen: D3.js oder Chart.js

## Installation
1. Klone das Repository:
   ```bash
   git clone https://github.com/deinbenutzername/klimadaten-analyse-tool.git
   ```
2. Wechsle in das Verzeichnis:
   ```bash
   cd klimadaten-analyse-tool
   ```
3. Installiere die Abhängigkeiten:
   ```bash
   npm install
   ```
4. Starte den Server:
   ```bash
   npm start
   ```
